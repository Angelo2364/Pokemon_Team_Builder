Aplicação web desenvolvida em React para montagem de equipes Pokémon com foco nos jogos principais. 

O projeto utiliza a PokéAPI para obter dados oficiais dos Pokémon, incluindo sprites, tipos, habilidades, golpes, cadeias evolutivas e informações gerais. O projeto tem muitas funcionalidades e eu espero que estejam todas funcionando.

As Pokédex regionais e exclusividades de versões são mantidas manualmente para reproduzir com precisão a disponibilidade dos Pokémon em cada jogo.

Objetivo

O objetivo do projeto é oferecer uma ferramenta simples e organizada para jogadores que desejam montar equipes, planejar jornadas em diferentes jogos da franquia Pokémon e consultar rapidamente quais Pokémon estão disponíveis em cada versão.
