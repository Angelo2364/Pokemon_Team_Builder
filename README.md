# Pokemon_Team_Builder
Aplicação web desenvolvida em React para montagem de equipes Pokémon com foco nos jogos principais.   O projeto utiliza a PokéAPI para obter dados oficiais dos Pokémon, incluindo sprites, tipos, habilidades, golpes, cadeias evolutivas e informações gerais. O projeto tem muitas funcionalidades e eu espero que estejam todas funcionando.
