# Pokemon_Team_Builder
Aplicação web desenvolvida em React para montagem de equipes Pokémon com foco nos jogos principais.   O projeto utiliza a PokéAPI para obter dados oficiais dos Pokémon, incluindo sprites, tipos, habilidades, golpes, cadeias evolutivas e informações gerais. O projeto tem muitas funcionalidades e eu espero que estejam todas funcionando. 
Funcionalidades
Montagem de equipes Pokémon com até 6 integrantes.
Busca rápida por nome.
Filtro por geração.
Filtro por tipo.
Filtro por estágio evolutivo.
Filtro por golpes disponíveis.
Filtro por habilidades.
Opção para ocultar Pokémon lendários.
Organização por Pokédex regional de cada jogo.
Separação por áreas da Pokédex (ex.: Central Kalos, Coastal Kalos e Mountain Kalos).
Sistema de exclusividades entre versões.
Exibição de sprites e informações básicas dos Pokémon.
Cache local para reduzir chamadas repetidas à API.
Suporte a formas especiais e correções de nomenclatura da PokéAPI.

As Pokédex regionais e exclusividades de versões são mantidas manualmente para reproduzir com precisão a disponibilidade dos Pokémon em cada jogo.

Objetivo

O objetivo do projeto é oferecer uma ferramenta simples e organizada para jogadores que desejam montar equipes, planejar jornadas em diferentes jogos da franquia Pokémon e consultar rapidamente quais Pokémon estão disponíveis em cada versão.
